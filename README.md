# wxyz
> has nothing to do with wx

[![binder-badge][]][binder]

Try to bring more isomorphic capabilities to the Jupyter Widget stack

| ipython |    widget    | js         |
| ------: | :----------: | ---------- |
|  jinja2 | **Template** | nunjucks   |
|    json |   **JSON**   | JSON       |
| mistune | **Markdown** | marked     |
|  pyyaml |   **YAML**   | js-yaml    |
|       - |  **Editor**  | CodeMirror |
|       - |   **Dock**   | Phosphor   |
