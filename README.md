> has nothing to do with wx

| ipython |    widget    | js         |
| ------: | :----------: | ---------- |
|  jinja2 | **Template** | nunjucks   |
|    json |   **JSON**   | JSON       |
| mistune | **Markdown** | marked     |
|  pyyaml |   **YAML**   | js-yaml    |
|       - |  **Editor**  | CodeMirror |
|       - |   **Dock**   | Phosphor   |
